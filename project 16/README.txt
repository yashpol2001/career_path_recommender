To run the application you can use the deployed link on render - “ https://cp-frontend-lbv0.onrender.com  ”  				
or 				Execute manually by  - running both backend and frontend folders separately on terminals 
- in /backend run - python3 -m uvicorn main:app --reload		
- in /frontend run - npm run dev                         
 after running both backend and front end - you can access the application through
- http://localhost:5173/
